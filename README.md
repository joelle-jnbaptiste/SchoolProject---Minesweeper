## ğŸ¯ About The Project

This project is a **Java implementation of the classic Minesweeper game**, developed as part of a school curriculum.

The main objective was to practice **core programming concepts**, including:

- Object-oriented programming (OOP)
- Game logic and state management
- Grid-based algorithms
- User interaction handling
- Win and loss condition detection

The game includes:

- A grid-based board
- Random mine placement
- Cell revealing logic
- Detection of winning and losing conditions

This project focuses on **algorithmic thinking and Java fundamentals** rather than advanced graphics or UI frameworks.

This project runs locally using a Java development environment.

- Java JDK (8 or higher)
- A Java-compatible IDE (IntelliJ IDEA, Eclipse, or similar)

### Installation

1. Clone the repository:

       git clone https://github.com/joelle-jnbaptiste/SchoolProject---Minesweeper.git
